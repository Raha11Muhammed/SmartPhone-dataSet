# Smartphone Dataset Analysis Project 📱

This project analyzes a smartphone dataset using Python and Jupyter Notebook. It demonstrates data analysis, statistical calculations, and data visualization techniques using popular Python libraries.

---

## Features
- Read and process CSV dataset
- Display dataset information
- Analyze smartphone specifications
- Calculate averages and statistical values
- Create charts and graphs
- Explore data distributions and trends

---

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- SciPy
- Jupyter Notebook

---

## Dataset
The dataset contains smartphone-related information used for statistical analysis and visualization.

### Dataset Columns
- Model Name
- Price
- Rating
- RAM
- Storage

---

## Project Files
- `Untitled1.ipynb` → Main Jupyter Notebook
- `cleaned_data_500x5.csv` → Smartphone dataset

## How to Run

1. Download the project files
2. Open the notebook in Jupyter Notebook or Google Colab
3. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn scipy
```

4. Run all notebook cells
